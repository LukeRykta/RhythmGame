# RowdyHacks2022
You will need JDK 1.8 to compile!
Built using Gradle. 

In your IDE, add a configuration using a java 8 SDK, 
set the classpath (-cp) to 'RowdyHacks2022.desktop.main,'
and select 'mygdx.game.DesktopLauncher' as your main class.
